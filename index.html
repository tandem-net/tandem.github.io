Step 01 · Install

Very easy download.

Pull Tandem down and run the installer. It drops the compile engine into ~/.tandem and wires up the CLI and Node system.

bash
$ ./install.shTandem compile engine installed at /home/wnusair/.tandem/bin/tandem-compile Run: tandem --help Note: 'tandem' currently resolves to /home/wnusair/Nextcloud/Summer Project (lockheed martin rizz)/Tandem/tandem-aio/venv/bin/tandem, which comes before /home/wnusair/.local/bin on your PATH. Adjust your PATH order if you want this install to win. Next steps: 1. Log in: tandem auth login 2. Start your node: tandem node start (or run it in the background: tandem node enable) 3. Check on it: tandem status Your node needs to be running before you can deploy or start a job.
Step 02 · Auth

Secure Authentication.

Simple authentication. Connect to our central server, or use a private / organization server; all available for free. Every task and job is tied to your account

Step 03 · Go live

Join the neighborhood pool.

Start a node and Tandem reserves your spare RAM, CPU, and RAM power to power the community of developers.

tandem
$ tandem node startTandem node started in the background. It keeps running after you close this terminal. Check on it any time with: tandem status
